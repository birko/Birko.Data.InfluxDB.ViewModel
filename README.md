# Birko.Data.InfluxDB.ViewModel

InfluxDB ViewModel repository implementations for the Birko Framework.

## Features

- ViewModel repositories for InfluxDB
- Health checks and drop operations
- Settings-based configuration

## Installation

```bash
dotnet add package Birko.Data.InfluxDB.ViewModel
```

## Dependencies

- Birko.Data.Core (AbstractModel, ViewModels)
- Birko.Data.Stores (store interfaces)
- Birko.Data.Repositories (repository interfaces)
- Birko.Data.ViewModel
- Birko.Data.InfluxDB
